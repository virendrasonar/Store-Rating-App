# 📌 Project Title  
**Store Rating Web Application**

---

## 📖 Overview  
This is a full stack web application that allows users to rate stores on a scale of 1 to 5.  
The system supports role-based access, where different users (Admin, Normal User, Store Owner) have different functionalities.

---

## 🚀 Features Implemented  
- User authentication (login system)  
- Role-based access control (Admin / User / Store Owner)  
- Store listing and rating system (1–5 scale)  
- Users can submit and update ratings  
- Basic backend APIs for core operations  
- Database integration (MySQL/PostgreSQL)
